# Vortex Operator Theory (VOT)

A dynamical resonance framework for modeling the zeros of the Riemann zeta function.

## What this repository is

This repository contains a research prototype exploring a non-classical approach to the zeros of the Riemann zeta function.

Core ideas:
- a prime-derived dynamical signal `A₀(t)`
- local time-frequency analysis
- a vortex ontology: **shell / 0-point / flow**
- event types: **anchor / bridge / transition**
- a **V1-backed hybrid** architecture:
  - V1 = global scaffold
  - precision = sharp local correction
  - rescue = flexible correction
  - local_rescue = targeted patch for hard clusters

## Status

**Research prototype**  
**Not a proof of the Riemann Hypothesis**

This repo is meant as a structured research program and preprint-style presentation, not as a final theorem.

## Current empirical findings

The project suggests:
- zeta zeros are not homogeneous
- multiple dynamical mechanisms appear to exist
- most low zeros behave as **shell_contact**
- a minority behave as **shell_approach**
- some hard clusters require **local_rescue**
- a **V1-backed hybrid** scales better than purely local flexible models
