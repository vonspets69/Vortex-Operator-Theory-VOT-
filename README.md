# Vortex Operator Theory (VOT)

A dynamical resonance framework for modeling the zeros of the Riemann zeta function.

## Overview

VOT models zeta zeros as outcomes of evolving resonance structures derived from a prime-based signal A₀(t).

Key concepts:
- multi-layer spectral structure
- vortex ontology: shell / 0-point / flow
- hybrid model: V1 backbone + local precision/rescue
