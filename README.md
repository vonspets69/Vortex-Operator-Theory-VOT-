# Vortex Operator Theory (VOT)

A dynamical resonance framework for modeling the zeros of the Riemann zeta function.

## What this is

This repository contains a research prototype exploring a non-classical approach to zeta zeros:

- dynamic resonance model based on prime-derived signal
- multi-layer spectral structure
- vortex interpretation (shell / 0-point / flow)
- hybrid model combining global scaffold and local corrections
